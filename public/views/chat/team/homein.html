About GF-chat

GF-chat is coming from an innovation project, internship program for four months at Rise Up Company. This project built from team internships as to test creative and innovative idea to improve their ability and technical skill from what they have learned at school during 2 years.


Project Purpose

GF-chat creates from many ideas and discussion in one purpose to make user know clearly about Gift and Freebies projects and enjoy with real chat with a developer about both projects.


Project Status

This project was developed for full more than 2 months. It has 2 feather is real chat and invite friends.


Technical in use

This project builds a base from Firebase as server side, and Anjularjs as client side. Furthermore, we also use bootstrap.
